Git常用命令整理

2020-04-13 22:32:20

Git 常用命令整理, 方便查阅使用

一、 git 基本命令

1. branch 命令

git branch newbranch //创建branch分支
git branch //显示本地分支
git branch -a // 显示远程分支

2. 切换分支

git checkout branchname // 切换到branchname分支
git checkout -b branchname // 创建并切换到branchname分支

3. 删除分支

git branch -d branchname //删除branchname分支
    -d 删除合并到主干的分支
    -D 强制删除branchname分支

4. 合并分支

git merge branchname

5. 撤销上一次的 commit

git revert HEAD

6. 撤销本地所有的修改

git reset --hard

7. 将上次 commit 撤回暂存区

git reset --soft HEAD^ //前两次就是HEAD^^
git revert // 撤销操作，操作之前的commit会被保留
git reset  // 撤销某次提交，此次之后的修改会被退回暂存区

8. 将本次更新并入上次 commit，不提交新的 commit

git commit -amend

二、git 常用命令

1. git tag

为版本打标签：git tag revert_version bbaf6fb5060b4875b18ff9ff637ce118256d6f20'

2. stash

git stash
    把本地所有修改放到暂存区
git stash pop
    把stash的修改 pop 出来

三、问题与解决方案

1. 提交合并失败

解决冲突并重新提交：git commit -am "new commit file"
